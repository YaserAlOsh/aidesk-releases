# AIDesk Landing Page

This folder is a self-contained static landing page for GitHub Pages.

## Move To A Separate Repo

1. Create a new GitHub repository for the public site.
2. Copy the contents of this `landing/` folder to the root of that repo.
3. Push to GitHub.
4. In the repo settings, enable GitHub Pages from the main branch root.

No build step is required.

## Files

- `index.html`: page content and SEO metadata.
- `styles.css`: responsive layout and visual design.
- `script.js`: mobile navigation and small browser enhancements.
- `assets/`: local images used by the page.
